Nametag

Enjoy!

First I found a file that was printable and uploaded it into Prusa Slicer. At this point it can be edited. If there is a color change then you will need to view it in the splitter, add it and then you will physically change the color of filament when ready. This was similar to things I have done in the past because I have had experience with editing the dimensions of objects in a CAM software. However, it was very different because I got to be creative with my color change and where it did to make my name tag a bit more than just 2 colors.

Stress Test

Enjoy!

This small cat was used as a stress test for the machine. At first glance it might not look like there is much information to gain, however the stringing on the cat is likely due to extra filament leaking from the extruder as it prints. This means that incredible details might be lost because extra filament might leak and consequently muddying the details. Furthermore there might be issues with speed because if the extruder is leaking, if it is moving faster it could likely mess up entire prints.

Assistive Divices

Enjoy!

This is a one handed reading device. It was made so that people with the inability to use both hands to whatever standard they personally hold for themselves, to read books with one hand and idealistically without assistance. I personally can’t speak to how effective this device is as it wasn’t designed for me, and as a able bodied person I believe it would do a disservice to those who need it to share an opinion that frankly has no stake here.
